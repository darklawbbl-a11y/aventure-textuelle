# TP01 — Aventure interactive : L’ornithorynque albinos

## Description
Ce projet est une histoire interactive sous forme de site web.  
On y suit un **ornithorynque albinos** qui se réveille dans une caverne mystérieuse et qui doit surmonter plusieurs embûches dans une forêt afin de trouver un lieu paisible où vivre.

## Instructions
- Ouvrir `index.html` dans un navigateur pour commencer l’histoire.  
- Cliquer sur les choix pour avancer dans l’aventure.  
- Les images sont placées dans `/images/`.  
- La musique est déposée dans `/music/`.
